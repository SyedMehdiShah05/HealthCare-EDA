# HealthCare-EDA

# Healthcare EDA & Patient Prediction System

## Project Overview

This project is based on Healthcare Exploratory Data Analysis (EDA) and patient condition prediction using a healthcare dataset. The system analyzes patient data and predicts whether a patient is Normal, Abnormal, or falls into another health category based on the dataset.

## Features

* Healthcare data analysis and visualization
* Data cleaning and preprocessing
* Patient condition prediction using Machine Learning
* Simple and user-friendly website interface
* Display of prediction results in real time

## Technologies Used

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-learn
* HTML, CSS, Flask

## Objective

The main goal of this project is to apply Data Science and Machine Learning techniques in the healthcare field to analyze patient data and build a simple prediction system.

## Conclusion

This project demonstrates the use of EDA, Machine Learning, and Web Development to create a healthcare prediction application with an easy-to-use interface.
